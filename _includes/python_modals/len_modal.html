len()

The len() evaluates how big an object is.

In this case of a string, this is how many characters. 

text = "fez"
len_of_text = len(text)
print(len_of_text)
                        
3

In the case of a list, it is how many elements there are in the list. 

myList = ["fez", "code"]
len_of_list = len(myList)
print(len_of_list)
                        
2
