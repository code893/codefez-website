2D Arrays

Lists are used to store multiple items in a single variable. A 2D array is when each item in a list is another list which contains it's own items. 




grid = [
  ["x","o","x"],
  ["o","x","o"],
  ["x","o","x"]
]
for r in grid:
  print(r)
