Inline For Loops

an inline for loop allows us to iterate a list of instructions for a specified number of times on 1 line. 

names = ["Alice", "Bob", "Charlie"]
upper_case = [name.upper() for name in names]
print(upper_case)
                
                  
    ALICE
    BOB
    CHARLIE

In this example, we have a list of names. We want to convert each name to uppercase. We can use an inline for loop to iterate over the list and convert each name to uppercase.
