Open

The open() function will take a filepath as an argument to open a file.
There are 2 methods to opening a file. Both methods will assign the open file to a variable.
Notice the 2 examples below have 2 arguments. The filename and then the access mode. 

# The open file is assigned to a variable
file = open("filename.txt", "r")

# Opened files should also be closed.
file.close()
                        
""" with this method, there is no need to 
close the file. """
with open("filename.txt", "r") as file:
    # add instructions here

Files should be opened with an access mode.

  • "r" is for opening a file only for reading it.
  • "w" is for overwriting all contents of the file.
  • "a" is for appending to a file.
